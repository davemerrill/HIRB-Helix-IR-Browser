Drop up to 8 Helix setlist files onto this page MORE INFO
IMPORTANT: They should all use the same set of IRs.

IRs are listed on the left, by slot number.
For each one, the number of presets in each setlist that use it and the total number of uses are shown.

Presets with IRs are listed on the right.
The two lists stack vertically on narrower screens.

Click anywhere in a row in the IRs list to select or deselect it.
That filters the presets list to show only ones that use at least one of the selected IRs.
Ctrl-click to deselect all other IRs, so only the one you click is selected.

Click the Copy button in either list to view the data in tab-tab-rtn format.
Copy and paste into a spreadsheet for further analysis or manipulation.
